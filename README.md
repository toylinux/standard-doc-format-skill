# 标准格式 Word 文档生成技能（standard-doc-format）

把符合中文公文 / 学校材料排版规范的 Word 文档生成流程，固化成一个可复用的 WorkBuddy 技能。
在对话中只要说「使用标准格式」，即可按以下规范自动生成 `.docx`。

## 适用场景

- 学校师德师风 / 培训感悟、工作总结、汇报材料等
- 任何需要「标准格式」中文文档的场合（标题方正小标宋简体二号居中、署名楷体、正文仿宋、一级黑体、二级楷体加粗、固定行距与页边距、居中页码）

## 格式规范

| 元素 | 字体 | 字号 | 对齐 / 缩进 |
|------|------|------|-------------|
| 标题 | 方正小标宋简体 | 二号 | 居中 |
| 署名 | 楷体_GB2312 | 三号 | 居中 |
| 一级标题 | 黑体 | 三号 | 首行缩进 2 字符 |
| 二级标题 | 楷体_GB2312（加粗） | 三号 | 首行缩进 2 字符 |
| 正文 | 仿宋_GB2312 | 三号 | 两端对齐、首行缩进 2 字符 |

- 页边距：上 / 下 2.54cm，左 / 右 3.18cm
- 行距：固定值 30 磅
- 页脚：居中纯数字页码（小四，12pt）

## 使用方法

在 WorkBuddy 中，只要说「使用标准格式」，即可触发本技能。技能脚本
`scripts/standard_doc.py` 读取一个**带标签的文本文件**，输出符合规范的 `.docx`。

### 输入格式（标签说明）

| 标签 | 含义 | 示例 |
|------|------|------|
| `[TITLE]` | 文档标题（单行） | `[TITLE] 如何做一名"四有"好教师` |
| `[SIGN]` | 署名（单行） | `[SIGN] 古塔区保二小学  李晓莉` |
| `[H1]` | 一级标题 | `[H1] 一、有理想信念` |
| `[H2]` | 二级标题 | `[H2] （一）立足课堂` |
| `[BODY]` | 正文段落 | `[BODY] 作为小学英语教师……` |

每段占一行，空行可省略。示例：

```text
[TITLE] 如何做一名"四有"好教师
[SIGN] 古塔区保二小学  李晓莉
[H1] 一、有理想信念
[BODY] 作为小学英语教师，我始终把育人放在首位……
[H2] （一）立足课堂主阵地
[BODY] 在课堂上，我用情境与游戏激发学生的学习兴趣……
[H1] 二、有道德情操
[BODY] 师德师风是教师的立身之本……
```

### 依赖

- Python 3 + `python-docx`（建议在受管虚拟环境中安装：`pip install python-docx`）

### 命令

```bash
python scripts/standard_doc.py <输入.txt> <输出.docx>
```

## 字体说明

「方正小标宋简体」为方正字库字体，若本机未安装，标题会回退为默认字体；
其余字体（楷体_GB2312、仿宋_GB2312、黑体）Windows 一般自带。需要保证标题原貌时，
本机安装「方正小标宋简体」即可。
